# StudyGo nr. Anki
Een StudyGo scraper die StudyGo woordenlijsten omzet tot Anki flitskaarten.

## Gebruik
Maak een urls.txt aan in dezelfde folder als de executable. In de urls.txt kunnen dan StudyGo links onderelkaar gevoegd worden. Daarna kan het programma gestart worden, na het draaien hoort er een anki.csv in de folder te zijn. Dit bestand kan geÃ¯mporteerd worden in Anki.
