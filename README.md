# StudyGo naar Anki
Een StudyGo scraper die StudyGo woordenlijsten omzet tot Anki flitskaarten. Bedoelt om de commercialiseren van onderwijs door half-gare diensten tegen te gaan. 

## Compileren 
```
cargo build --release
```

## Gebruik
Maak een urls.txt aan in dezelfde folder als de executable. In de urls.txt kunnen dan StudyGo links onderelkaar gevoegd worden. Daarna kan het programma gestart worden, na het draaien hoort er een anki.csv in de folder te zijn. Dit bestand kan vervolgens makkelijk ge√Ømporteerd worden in Anki.
